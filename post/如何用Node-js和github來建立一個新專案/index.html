如何用Node.js和github來建立一個新專案

Oct 26, 2012

要使用node.js必須先建立它的開發環境，下載Node.js之後把它安裝起來，你在你的command line裡面打node -v應該就會出現版本號了，node.js就這樣裝好了。

node.js有內建套件管理系統npm，你可以利用npm安裝各式各樣的前人開發的node.js外掛

接下來要選擇framework，現在node.js上最多人用的MVC framework就是express，node.js的套件都是經由npm安裝的，安裝的方法如下：

npm install express -g
