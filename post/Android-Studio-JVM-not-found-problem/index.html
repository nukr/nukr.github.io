Android Studio JVM not found problem

Dec 28, 2014

Android Studio 實在很爛，遲早把他換掉，在換掉他之前還是要讓他能動，還是得要解決這個問題

vi /Application/Android\ Studio.app/Contents/Info.plist

找到 1.6* 這個字串，把它改成 1.8* 即可。
