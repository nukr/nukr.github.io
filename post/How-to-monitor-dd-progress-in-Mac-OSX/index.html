How to monitor dd progress in Mac OSX

Apr 27, 2015 
brew install pv
sudo dd /dev/urandom | pv | sudo dd of=/dev/null

輸出

1,74MB 0:00:09 [ 198kB/s] [      <=>                            ]

也可以這樣用

pv /home/user/bigfile.iso | md5sum
